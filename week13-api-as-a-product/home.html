Giới thiệu

Chào mừng đến với UET-FA API. Chúng tôi cung cấp giải pháp AI để trả lời tự động các câu hỏi của sinh viên về quy chế đào tạo, học phí, và lịch sinh hoạt.

Tích hợp UET-FA vào website trường hoặc ứng dụng mobile của bạn chỉ trong vài phút.

Tài liệu API (Documentation)

Base URL: https://api.uetfa.com/v1

Endpoint: Gửi câu hỏi tư vấn

POST /chat/message

Mô tả: Gửi câu hỏi của sinh viên và nhận về câu trả lời từ AI.

Headers:

  • Authorization: Bearer YOUR_API_KEY
  • Content-Type: application/json

Body (JSON Request):

{
  "student_id": "SV23051546",    // (Tuỳ chọn) Mã sinh viên để cá nhân hóa
  "message": "Hạn đóng học phí học kỳ này là bao giờ?"
}

Response (JSON):

{
  "status": "success",
  "data": {
    "answer": "Theo thông báo số 123/TB-DT, hạn đóng học phí học kỳ 1 năm 2025 là ngày 5/12/2025.",
    "confidence": 0.98,
    "source": "Phòng Tài Chính - Kế Toán"
  }
}
